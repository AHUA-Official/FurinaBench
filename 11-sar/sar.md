# SAR 简介

SAR（System Activity Reporter，系统活动报告器）是 Linux 系统中常用的性能监控工具，用于收集、报告和保存系统活动信息。

## 主要功能

SAR 可以监控系统的多个方面：

- **CPU 使用情况**：用户、系统、空闲时间等
- **内存使用**：物理内存、虚拟内存、缓冲区等
- **磁盘 I/O**：读写速率、队列长度、利用率等
- **网络活动**：网络接口流量、错误统计等
- **进程活动**：进程创建、上下文切换等
- **文件系统**：inode 使用、文件打开数等

## 特点

- **历史数据**：通过 sysstat 包可以自动收集和保存历史性能数据
- **实时监控**：可以实时查看当前系统状态
- **灵活报告**：支持多种时间间隔和统计选项
- **轻量级**：对系统性能影响很小

## 常用命令

```bash
# 查看当前 CPU 使用情况
sar -u 1 5

# 查看内存使用情况
sar -r 1 5

# 查看磁盘 I/O
sar -d 1 5

# 查看网络统计
sar -n DEV 1 5

# 查看历史数据（需要 sysstat 配置）
sar -u
```
